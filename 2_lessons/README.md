# Раздел 2. Углубленные уроки

Это индекс второй линии материалов.
Здесь мы держим не короткие сценарии уроков, а более развернутые книги, учебные траектории и прикладные playbooks по работе с агентами.

## Что здесь лежит

### Конспекты траекторий

- [codex-agent-learning-path.md](./codex-agent-learning-path.md) — короткий маршрут освоения Codex.
- [claude-code-agent-learning-path.md](./claude-code-agent-learning-path.md) — короткий маршрут освоения Claude Code.

### Полные книги

- [codex-book/README.md](./codex-book/README.md) — полная книга по Codex.
- [claude-code-book/README.md](./claude-code-book/README.md) — полная книга по Claude Code.

### Практический сравнительный гид

- [codex-vs-claude-code-in-practice.md](./codex-vs-claude-code-in-practice.md) — практический гид по выбору между Codex и Claude Code на реальных рабочих сценариях.

### Прикладные playbooks

- [playbooks/README.md](./playbooks/README.md) — набор готовых рабочих сценариев.
Сейчас в нем уже есть сценарии от первого выбора инструмента до недельного производственного цикла, сборки проектных инструкций, безопасного роста автономности и минимальной организации работы маленькой команды.

## Как читать этот раздел

Если вы только входите в тему, лучше идти так:
1. сначала открыть нужный конспект траектории;
2. потом перейти в соответствующую полную книгу;
3. затем перейти в [playbooks/README.md](./playbooks/README.md) и пройти несколько базовых практических сценариев;
4. после этого открыть [codex-vs-claude-code-in-practice.md](./codex-vs-claude-code-in-practice.md) и сравнить две среды уже на понятной рабочей базе.

## Быстрый выбор

- Если вы хотите сначала понять экосистему OpenAI и Codex, начните с [codex-book/README.md](./codex-book/README.md).
- Если вам ближе экосистема Anthropic и Claude Code, начните с [claude-code-book/README.md](./claude-code-book/README.md).
- Если вы уже поняли основы и хотите готовые сценарии работы, идите в [playbooks/README.md](./playbooks/README.md). Там уже есть ветка не только для первых шагов, но и для регулярного недельного производства материалов, сборки устойчивых проектных инструкций, безопасного роста автономности и минимальной агентной организации.
- Если вы уже хотите не просто читать, а осознанно выбрать основную среду, откройте [codex-vs-claude-code-in-practice.md](./codex-vs-claude-code-in-practice.md).
- Если вы пока не уверены, откройте сначала оба коротких конспекта и выберите более понятную линию.
