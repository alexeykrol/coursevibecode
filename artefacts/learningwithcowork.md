# Как учиться с CoWork в 1000 раз быстрей?

Субботний эксперимент: дал Claude Cowork задачу и ушёл по делам. От старта до финиша почти 6 часов 100% автономной работы.

## Задача:

взять три платных курса на Udemy по Claude Code, Claude Cowork и Claude в MS Office, и извлечь из них всё содержание в структурированный Markdown. Транскрипты видеолекций, тексты статей, метаданные, заметки. Каждая лекция это отдельный файл по шаблону. Плюс индекс, конспект-саммари по каждому курсу, общий README.

## Что было на входе: 

три курса, 72 лекции, ~9.5 часов учебного видео и статей.

## Что на выходе:

— 72 Markdown-файла с полными транскриптами
— 3 индексных файла с оглавлением и описанием каждой лекции
— 3 конспекта-саммари (тезисная выжимка всего курса с перекрёстными ссылками)
— Обновлённые README.md и CLAUDE.md
— ~400 000 символов извлечённого и обработанного текста

## Сколько времени заняло:

~6 часов от первого сообщения до последнего файла.

Моё участие по сути минут 10: дал ссылки на курсы, пару раз подтвердил действия, в конце проверил результат.

Остальное полностью автономная работа агента.

## Как это работало?


1. Claude через браузерное расширение (Claude in Chrome) открывал Udemy, ходил по по лекциям, активировал панель транскриптов, извлекал текст через DOM-запросы.

2. Когда на третьем курсе видеоплеер отказался загружаться (Udemy требует активную вкладку), агент самостоятельно нашёл обходной путь — обнаружил внутреннее API Udemy, получил ссылки на VTT-файлы субтитров и скачал все транскрипты одним батч-запросом за секунды. Никто его этому не учил.

Что меня впечатлило больше всего: не объём работы (хотя 72 файла немало), а способность самостоятельно решать проблемы.

Видеоплеер не грузится → пробует кликнуть → не работает → пробует перезагрузить → не работает → находит API → качает напрямую. Это не скрипт по инструкции. Это адаптация.

Для контекста: если бы я делал это руками (открывал каждую лекцию, копировал транскрипт, форматировал в Markdown, писал заметки) это заняло бы дня 3-4 полноценной работы.

Мы уже в эпохе, когда AI не просто отвечает на вопросы. Он работает. Автономно, часами, решая проблемы по ходу дела.🤣

## До начала. Подготовка.

1. Иметь подписку на Claude - хотя бы за 20$
2. поставить в Chrome расширение от Claude (бесплатно)
3. подключить расширение к вашему аккаунту Claude
4. скачать десктопное приложение Claude (включая CoWork) - и подключить его к аккаунту Claude
5. Если это платный доступ типа Udemy, то взять подписку - это примерно 30$. Если какой-то б/п, то залогиниться и т.п.
6. Создать папку на диске (имя вашей папки для курсов)


## Создать файлы - см ниже и положить их в правильные места.

```
/имя вашей папки для курсов/
  CLAUDE.md                              ← This file (project instructions)
  .claude/
    settings.local.json                  ← Claude Code / Cowork permissions
  
```

## Процесс

1. Найти все нужные вам курсы, к примеру 100 или 300. Скачать файл со всеми ссылками.
2. Явно зайти на все эти курсы, у вас откроется ссылка, на которой уже Claude Cowork сможет работать.
3. Зайти в Cowork, попросить его подключить Chrome, т.е. чтобы он поднял отдельное окно и таб.
4. Скормить ему файл ссылками на все курсы.
5. Создать папку, куда Claude будет сгружать курсы.
6. Указать CoWork эту папку для работы, чтобы он понимал, куда все грузить.
7. Скачать мои настроечные инструкции (CLAUDE.md/ setting.json) и запустить пусковой файл, который развернет нужные инструкции в папке ваших курсов.
8. Сказать CoWork, чтобы он прочитал CLAUDE.md и начал работать.
9. Идти спать. В зависимости от вашего тарифа он будет делать примерно 1,5 часа курса/час.
10. На выходе - папка полностью скачанных курсов на английском. Потом просите его все перевести на русский, если надо.
