## Урок 1: IDE — Ваш умный рабочий стол

### **Продающий заголовок:**
**«Как превратить хаос файлов в умный офис, где всё находится за 2 секунды»**

### **Аннотация:**
Прямо сейчас ваши файлы разбросаны по папкам, вы открываете их в разных программах, теряете изменения, путаетесь в версиях. Вы тратите 30-40% времени не на работу, а на **поиск, открытие, переключение**.

В этом уроке вы получите **умный рабочий стол** — IDE (Integrated Development Environment), который станет вашим центром управления всеми проектами.

**Что изменится конкретно:**

- **Мгновенный поиск:** Вы получите поиск по всем файлам проекта за доли секунды (попробуете прямо в уроке)
- **Всё в одном окне:** Вы откроете 20 файлов одновременно и будете видеть их структуру (создадите свое первое рабочее пространство)
- **Умная навигация:** Вы получите возможность прыгать между связанными частями проекта одной кнопкой (выполните 5 упражнений)
- **Предпросмотр всего:** Вы увидите содержимое файлов, не открывая их (настроите за 2 минуты)

**Это НЕ программа для кода.** Это умный органайзер для работы с любыми текстовыми проектами — от курсов до исследований, от документов до книг.

**Что вы получите прямо сейчас:**
- Установленный VS Code на вашем компьютере (3 минуты)
- Ваш первый проект, открытый в IDE (5 минут)
- Настроенный интерфейс под вашу работу (10 минут)
- Чек-лист из 10 горячих клавиш для мгновенной навигации (начнете использовать сегодня)

## Урок 2: Git — Машина времени для ваших проектов

### **Продающий заголовок:**
**«Как никогда больше не потерять ни одного изменения и вернуться к любой версии за 10 секунд»**

### **Аннотация:**
Сколько раз вы удаляли важный текст, а потом не могли вспомнить, что там было? Сколько раз вы делали изменения, которые всё сломали, и не могли откатиться назад? Сколько раз вы создавали файлы типа «версия_финал_2_окончательная_правки.doc»?

В этом уроке вы получите **машину времени** для всех ваших проектов — систему контроля версий Git.

**Что изменится конкретно:**

- **История каждого изменения:** Вы получите полную историю всех правок с возможностью вернуться к любой версии (сделаете первый коммит прямо в уроке)
- **Безопасные эксперименты:** Вы сможете пробовать любые изменения, зная, что всегда вернетесь назад (создадите и удалите ветку за 30 секунд)
- **Понимание решений:** Вы увидите, почему вы приняли решение 3 месяца назад (напишете первое commit-сообщение)
- **Работа без страха:** Вы перестанете бояться что-то сломать или потерять (выполните упражнение с откатом)

**Аналогия:** Это как Ctrl+Z, но для всего проекта, навсегда, с объяснением каждого шага.

**Что вы получите прямо сейчас:**
- Установленный Git на вашем компьютере (2 минуты)
- Ваш первый репозиторий с историей изменений (5 минут)
- Навык делать "снимки" проекта (commit) — выполните 3 раза
- Навык откатываться назад — попробуете вернуться к вчерашней версии
- Чек-лист из 5 команд Git, которые будете использовать каждый день

## Урок 3: GitHub — Надежный сейф в облаке

### **Продающий заголовок:**
**«Как сделать резервную копию всей жизни и работать с любого устройства в любой точке мира»**

### **Аннотация:**
Ваш компьютер может сломаться. Файлы могут пропасть. Диск может отказать. А вместе с ним — годы работы, проекты, знания. Вы рискуете **всем** каждый день.

В этом уроке вы получите **надежный сейф в облаке** — GitHub, где все ваши проекты будут храниться в безопасности и доступны с любого устройства.

**Что изменится конкретно:**

- **Абсолютная безопасность:** Вы получите автоматическое резервное копирование всех проектов (создадите первый remote-репозиторий)
- **Работа отовсюду:** Вы сможете открыть проект на ноутбуке, продолжить на планшете, проверить с телефона (синхронизируете устройства)
- **Портфолио и репутация:** Вы получите публичное портфолио всех ваших работ (создадите профиль за 2 минуты)
- **Совместная работа:** Вы сможете работать с другими людьми над одним проектом без конфликтов (попробуете пригласить коллегу)

**Важно:** GitHub — это не только для программистов. Это надежное облако для любых проектов: курсов, книг, исследований, документации.

**Что вы получите прямо сейчас:**
- Созданный аккаунт на GitHub (3 минуты)
- Ваш первый проект, сохраненный в облаке (5 минут)
- Настроенную синхронизацию между компьютером и облаком (выполните push и pull)
- Навык восстановления проекта с нуля — удалите локальную копию и верните из облака
- Чек-лист из 3 команд для ежедневной синхронизации

## Урок 4: AI-Агент — Исполнитель, который работает за вас

### **Продающий заголовок:**
**«Как получить сотрудника, который видит весь проект, понимает контекст и делает работу за вас»**

### **Аннотация:**
Чат — это консультант. Он дает советы, но **вы** делаете всю работу: копируете, вставляете, проверяете, исправляете. Часами.

В этом уроке вы получите **настоящего исполнителя** — AI-агента, который работает прямо в вашей IDE и делает работу **вместо вас**.

**Что изменится конкретно:**

- **Агент видит весь проект:** Вы дадите задание на 100 файлов — агент найдет и исправит всё сразу (попробуете на своем проекте)
- **Агент помнит контекст:** Вы продолжите работу через неделю — агент вспомнит всё (проверите на практике)
- **Агент делает, а не советует:** Вы скажете "измени это в 50 местах" — агент сделает за 30 секунд (выполните реальную задачу)
- **Агент работает в вашей среде:** Никакого копипаста — агент создает, редактирует, удаляет файлы сам (увидите результат)

**Ключевое отличие:**
- Чат: «Вот код, скопируй его и вставь сам»
- Агент: «Задача выполнена, файлы обновлены, хотите увидеть изменения?»

**Что вы получите прямо сейчас:**
- Установленный AI-агент (Cline/Claude Code) в вашей IDE (5 минут)
- Ваше первое выполненное задание агентом — создание или правка файлов (10 минут)
- Навык ставить задачи агенту — выполните 3 разных типа задач
- Навык проверять работу агента — научитесь контролировать результат
- Чек-лист из 7 типов задач, которые агент делает лучше всего

## Урок 5 (Опционально): Собираем всё вместе — Ваш цифровой офис

### **Продающий заголовок:**
**«Как за 30 минут настроить систему, которая изменит вашу работу навсегда»**

### **Аннотация:**
Теперь у вас есть все инструменты. Но инструменты — это не система. В этом уроке вы получите **работающую систему** — ваш полноценный цифровой офис.

**Что вы настроите:**

1. **Единое рабочее пространство:** Вы создадите шаблон проекта, который будете использовать для всех работ (получите готовый шаблон)
2. **Автоматический workflow:** Вы настроите автоматическое сохранение, синхронизацию, бэкапы (настроите за 10 минут)
3. **Связку всех инструментов:** IDE + Git + GitHub + AI-агент заработают как единый механизм (проверите на реальном проекте)
4. **Контрольный чек-лист:** Вы получите ежедневный чек-лист работы в новой среде (распечатаете и повесите на стену)

**Практическое задание:** Вы создадите свой первый полноценный проект от начала до конца с использованием всех 4 инструментов.

**Что вы получите прямо сейчас:**
- Полностью настроенную рабочую среду (30 минут)
- Шаблон проекта для повторного использования
- Ваш первый проект, созданный в новой системе
- Понимание, как работать дальше самостоятельно
- Чек-лист для ежедневной работы (5 минут каждое утро)
