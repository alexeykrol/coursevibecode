# Итоговая структура курса: «Вайб-кодинг и AI-агенты»

**Статус:** 🟢 Утверждено к разработке  
**Дата:** 31.01.2026  
**Концепция:** Курс как игра — последовательность уроков (игровых циклов) без деления на блоки

---

## Ключевые принципы структуры

1. **Нет блоков — только уроки**: Каждый урок = игровой цикл (возможность → инструмент → действие → приз)
2. **Вводные уроки = игровой цикл**: Вводная часть тоже построена по игровой механике
3. **Постепенная детализация**: По ходу работы будем корректировать и дополнять
4. **Прогрессия сложности**: От простого к сложному, от чата к профессиональной среде

---

## ПОСЛЕДОВАТЕЛЬНОСТЬ УРОКОВ

### 🎯 Вводные уроки (Игровой пролог)

**Урок 1: Туристический маршрут — что тебя ждет**
- **Возможность:** Ты увидишь весь путь от чата до создания своих приложений
- **Инструмент:** Карта курса, обещания, мотивация
- **Действие:** Пройди тест "Какой ты сейчас и кем станешь"
- **Приз:** Понимание своего пути и мотивация начать

**Урок 2: Две революции прямо сейчас**
- **Возможность:** Ты получишь 2 суперсилы: скорость x10 + возможности программиста
- **Инструмент:** Понимание "производительность + возможности"
- **Действие:** Посчитай экономию времени на своих задачах
- **Приз:** Конкретные цифры твоего выигрыша (часы/деньги)

**Урок 3: От аренды к владению — почему это меняет класс жизни**
- **Возможность:** Ты создашь активы, которыми владеешь на 100%
- **Инструмент:** Понимание суверенитета vs аренды
- **Действие:** Оцени стоимость своего будущего актива
- **Приз:** Калькулятор потенциальной стоимости + мотивация

### 🎮 Уровень 1: Вайб-кодинг в чате (примитивный)

**Урок 4: Твой первый проект за 10 минут — в чате**
- **Возможность:** Создай простое приложение прямо сейчас, без установки
- **Инструмент:** ChatGPT (или Claude)
- **Действие:** Создай калькулятор/landing page/простую игру в чате
- **Приз:** Твой первый рабочий проект (но ты увидишь ограничения)

**Урок 5: Почему чат — это тупик (на практике)**
- **Возможность:** Ты увидишь 3 смертельных ограничения чата
- **Инструмент:** Диагностический чек-лист
- **Действие:** Попробуй решить сложную задачу в чате (провалишься)
- **Приз:** Понимание, что нужен следующий уровень

---

### 🎮 Уровень 2: Bolt и Lovable (промежуточный)

**Урок 6: Твой первый проект в Bolt — быстрее и мощнее**
- **Возможность:** Создай полноценное приложение с UI за 15 минут
- **Инструмент:** Bolt.new
- **Действие:** Создай todo-app или простой сайт с интерактивом
- **Приз:** Рабочее приложение, которое можно показать

**Урок 7: Lovable — следующий шаг**
- **Возможность:** Получи еще больше возможностей (база данных, аутентификация)
- **Инструмент:** Lovable
- **Действие:** Добавь функциональность к предыдущему проекту
- **Приз:** Приложение с backend-функциями

**Урок 8: Сравнение: чат vs Bolt vs Lovable**
- **Возможность:** Ты увидишь рост своих возможностей
- **Инструмент:** Сравнительная таблица + твой опыт
- **Действие:** Реши одну задачу во всех трех средах
- **Приз:** Понимание, когда что использовать (но limits всё еще есть)

### 🎮 Уровень 3: Профессиональная среда (VS Code + Git + GitHub + Агенты)

**Урок 9: VS Code — твой умный рабочий стол**
- **Возможность:** Работай с проектами любой сложности
- **Инструмент:** VS Code
- **Действие:** Установи, открой проект, освой навигацию
- **Приз:** Центр управления всеми проектами

**Урок 10: Git — машина времени**
- **Возможность:** Никогда не потеряй ни одного изменения
- **Инструмент:** Git
- **Действие:** Сделай commit, измени файл, откатись назад
- **Приз:** Свобода экспериментировать без страха

**Урок 11: GitHub — облачный сейф**
- **Возможность:** Твои проекты в безопасности + работа отовсюду
- **Инструмент:** GitHub
- **Действие:** Создай репозиторий, отправь в облако, восстанови
- **Приз:** Абсолютная безопасность данных

**Урок 12: AI-Агент — твой исполнитель**
- **Возможность:** Работа без копипаста, агент делает всё сам
- **Инструмент:** Cline (или другой агент)
- **Действие:** Дай первое задание, увидь как агент работает
- **Приз:** Первая задача решена в 10 раз быстрее

**Урок 13: Первая победа — реальный проект**
- **Возможность:** Реши задачу из своей работы
- **Инструмент:** Вся связка (VS Code + Git + GitHub + Агент)
- **Действие:** Возьми свой проект, реши реальную задачу
- **Приз:** Конкретная экономия времени на твоих цифрах

## Ключевые отличия от старой структуры

**Было:**
- Блоки: ЗАЧЕМ (мотивация) → ЧТО (инструменты) → КАК (практика) → ВСТРАИВАНИЕ

**Стало:**
- Уроки: каждый = игровой цикл (возможность → инструмент → действие → приз)
- Прогрессия: вводный → чат → Bolt/Lovable → профессиональная среда → проекты → монетизация
- Никаких разрывов между мотивацией и практикой

---

## Принципы работы над курсом

1. **Итеративность:** По ходу работы будем корректировать уроки
2. **Каждый урок отдельно:** Фокус на одном уроке за раз
3. **Игровая механика:** Постоянное ощущение прогресса
4. **Акцент на "получить + применить":** Никакого "узнаете" — только "получите"
5. **Мотивация на каждом шаге:** Продавать желание идти дальше
