# Урок 5: Почему чат — это тупик (на практике)

**Длительность:** 8-10 минут  
**Формат:** Скринкаст + разбор кейсов  
**Ключевой месседж:** Ты увидишь 3 реальных провала в чате — и поймешь, что пора двигаться дальше

---

## ХУК [00:00-00:20]
«Проект на 50 тысяч рублей. 3 дня работы. Всё развалилось за 20 минут. Сейчас покажу, как чат убивает серьезные проекты.»

## ОБЕЩАНИЕ [00:20-00:50]
- Кейс 1: Потерянный контекст (проект из 5 файлов)
- Кейс 2: Бесконечный рефакторинг (переписали 7 раз)
- Кейс 3: Невозможность отката (всё сломалось, версии нет)
- Чек-лист «Признаки тупика»

## КЕЙС 1: ПОТЕРЯННЫЙ КОНТЕКСТ [00:50-03:00]
**Ситуация:** Landing page + форма + отправка email + база данных

**Что пошло не так:**
- Файл 1 (HTML) работает
- Файл 2 (CSS) работает
- Файл 3 (JS для формы) — чат забыл структуру HTML
- Файл 4 (бэкенд) — не связывается с формой
- Файл 5 (база данных) — несовместим с бэкендом

**Проблема:** Чат видит только 1-2 последних файла. Остальные забывает.

**Время потеряно:** 8 часов на координацию

## КЕЙС 2: БЕСКОНЕЧНЫЙ РЕФАКТОРИНГ [03:00-05:00]
**Ситуация:** Простое приложение-калькулятор, усложнение

**Цикл:**
1. Чат дает код
2. Находим баг
3. Чат исправляет → ломается другое место
4. Исправляем → ломается третье
5. Чат предлагает: «Давайте перепишем с нуля»
6. Переписываем → те же проблемы
7. Повторяем цикл

**Результат:** 7 полных переписываний, 0 результата

**Время потеряно:** 12 часов

## КЕЙС 3: НЕВОЗМОЖНОСТЬ ОТКАТА [05:00-07:00]
**Ситуация:** Рабочее приложение, решили «улучшить дизайн»

**Что произошло:**
- Чат предложил новый CSS
- Вставили → всё сломалось
- Старая версия не сохранена
- Чат не помнит, как было
- Начинаем с нуля

**Время потеряно:** Весь проект (3 дня работы)

## ЧЕК-ЛИСТ «ПРИЗНАКИ ТУПИКА» [07:00-08:30]
**10 признаков, что пора выходить:**

1. Больше 3 файлов в проекте
2. Чат начинает противоречить сам себе
3. Исправление одного → ломает другое
4. Фраза «давайте перепишем с нуля» появилась 2+ раза
5. Боишься что-то менять
6. Копипаст занимает больше времени, чем работа
7. Нет понимания, какая версия работает
8. Нужно показать чату весь контекст каждый раз
9. Проект застрял больше чем на 2 дня
10. Чувствуешь, что теряешь контроль

**Если 3+ признака = пора на следующий уровень**

## ПРИЗ [08:30-09:00]
Ты увидел границу. Чат — для простых задач. Для серьезных проектов нужны инструменты профессионалов.

## CTA [09:00-09:30]
Урок 6: Bolt.new — промежуточная станция. Больше возможностей, меньше проблем. Но всё еще в браузере.
