## 4 элемента:

### IDE

Очень просто. IDE — это **одна программа, в которой сразу есть всё для работы над проектом**.

Вместо того чтобы:

— писать в одном месте,

— сохранять в другом,

— запускать в третьем,

— искать ошибки в четвёртом,

ты открываешь **одну программу**, и там всё собрано.

**Бытовой пример**

IDE — это как **кухня**, а не набор отдельных приборов.

Плита, духовка, ножи и мойка — в одном пространстве, а не разбросаны по квартире.

**Что IDE делает на практике**

— помогает писать (подсказывает, подсвечивает);

— сразу показывает ошибки;

— запускает проект;

— хранит структуру файлов;

— дружит с Git и GitHub.

#### **Для не-разрабов**

IDE — это:

— для текста сложнее Word,

— для проектов сложнее папки с файлами,

— для логики сложнее Excel.

Примеры IDE:

**Visual Studio Code**,

**IntelliJ IDEA**,

**Xcode**.

Если одной фразой:

**IDE — это рабочий стол для сложных цифровых проектов.**

Уровень уверенности: **95%**.

А чем отличиается от Notion, Coda, Obsidian?

Супер-вопрос. Коротко и по-человечески.

**IDE** и **Notion / Coda / Obsidian** решают **разные задачи**, хотя внешне похожи (окна, файлы, панели).
