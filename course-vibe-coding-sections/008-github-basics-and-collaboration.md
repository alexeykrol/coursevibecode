### **Без GitHub (только Git)**

Ты ведёшь историю проекта **у себя на компьютере**.

Это как:

— дневник, который лежит у тебя дома;

— если ноут сломался — дневник пропал;

— если хочешь показать кому-то — надо пересылать файлы.

**Пример:**

Ты пишешь книгу. Git хранит версии глав.

Хочешь поработать с другого компьютера или показать редактору — начинаются копии, флешки, почта.

---

### **С GitHub (Git + GitHub)**

GitHub — это **общее хранилище в интернете**.

Это как:

— положить дневник в банковскую ячейку;

— открыть доступ другим;

— всегда иметь последнюю версию под рукой.

**Примеры:**

- **Ты один, но с двумя компьютерами**

Дом + ноутбук.

GitHub — точка синхронизации. Всё всегда актуально.

- **Ты и редактор**

Ты правишь текст → редактор оставляет правки → всё видно и не теряется.

- **Ты и команда**

Каждый работает над своим кусочком, потом всё собирается вместе без хаоса.

- **Публичная витрина**

Хочешь показать проект миру — даёшь ссылку, а не архив.

---

### **В одной строке**

**Git — это как сохранять версии у себя.**

**GitHub — это хранить их в общем, безопасном месте.**

Пример сервиса: **GitHub**.

Уровень уверенности: **95%**.

- ИИ (агент) - Чем ИИ кодовый агент отличается от обычного ИИ чата?

Коротко, приземлённо и без технарщины.

**ИИ-чат** и **кодовый агент** — это **разные роли**, даже если внутри одна и та же модель.
