### **Наука и исследования (очень конкретно)**

1. **Обзор литературы (systematic review)**
    
    Статьи как файлы, агент сравнивает, ищет противоречия, обновляет выводы при добавлении новых работ.
    
2. **Формирование гипотез**
    
    Гипотезы, допущения, аргументы — агент следит за логической непротиворечивостью.
    
3. **Научные статьи и препринты**
    
    Черновики, версии, правки рецензентов — всё прозрачно и откатно.
    
4. **Методологии экспериментов**
    
    Протоколы, условия, параметры — агент проверяет согласованность.
    
5. **Интерпретация результатов**
    
    Таблицы, описания, выводы — агент помогает выявлять логические разрывы.
    
6. **Междисциплинарные исследования**
    
    Разные языки дисциплин — агент помогает свести в одну систему.
    
7. **Исследовательские дневники**
    
    Логи идей, наблюдений, ошибок — агент помогает извлекать инсайты.
    
8. **Репликация исследований**
    
    Агент сверяет методики и результаты между версиями.
    
9. **Грантовые заявки**
    
    Цели, новизна, метод, impact — агент держит логику заявки целиком.
    
10. **Научные коллаборации**
    
    Несколько авторов, правки, версии — Git как основа коллективного мышления.
